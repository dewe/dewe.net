WSS 3.0 för nedladdning

Javisst, nu är WSS 3.0 live och tillgängligt för nedladdning:
  • Windows SharePoint Services 3.0
  • Windows SharePoint Services 3.0 Language Pack
  • Windows SharePoint Services 3.0 x64 (64 bit)
  • Windows SharePoint Services 3.0 Language Pack x64 (64 bit)
Än så länge finns inte alla språkpaket, men det kommer efterhand.
